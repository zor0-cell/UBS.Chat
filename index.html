Настройка общей базы данных
Чтобы сообщения были видны всем пользователям в реальном времени, подключи бесплатную Firebase Realtime Database. Займёт ~3 минуты.
1
Открой console.firebase.google.com → нажми «Создать проект» → введи любое название → отключи Google Analytics → нажми «Создать проект».
2
В левом меню выбери Realtime Database → «Создать базу данных» → выбери любой регион → выбери режим «Тестовый» (test mode) → нажми «Готово».
3
Скопируй URL базы данных — выглядит так:
https://название-default-rtdb.firebaseio.com
Вставь его ниже и нажми «Подключить».
Данные хранятся в твоей Firebase — Anthropic/Claude не имеет к ним доступа.
Никаких логов создателя чата нет.
